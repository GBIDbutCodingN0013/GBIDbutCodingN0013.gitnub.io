如何让你的老旧Eclipse稍微好使一点

前言

最近找了家新公司实习，技术栈比较老了，开发用的Eclipse还是Neon的版本，本文是我个人的一些设置，虽然可能还是有些不好使，但是起码比刚开始装下来时好用多了

搞定 JPA project Change Event Handler

我在Build或者是切换开发环境的时候老是遇到 JPA project Change Event Handler 在progress里面堆了一堆，然后整个Eclipse卡死的问题，后来在网上搜了一下，把这个玩意卸载一下就好了
Dali Java Persistence Tools -JPA Support
如果还有 JPA project Change Event Handler 出现，就在安装目录下plugins/features下面把org.eclipse.jpt.*   删除再重启即可

搞定主题

个人感觉原生Eclipse的主题不是那么好看（黑暗模式），如果是换成白的话感觉刺得眼睛疼，所以我换成了第三方的主题，个人感觉很舒服
直接去Eclipse Marketplace搜索 devstyle，安装，重启，完事
